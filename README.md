# 简易图片处理工具 · ImageProcessor

一个不折腾的桌面图片处理小工具。  
不想学 PS，又想做点简单的图。  
就用它。

---

## 这个项目是干嘛的？

一句话：  
打开一张图，调一调，导出来。就这么简单。

适合这些场景：

- 改大小，发朋友圈、发简历、发报名表
- 加个文字水印，写个名字，防止被乱用
- 批量处理一堆图片，统一尺寸、统一格式、统一命名
- 像 PPT 那样放映一整个图片文件夹
- 想要“有点像 PS”，但又不想学 PS

---

## 功能一览

说重点，只列真正能用的：

- 图片打开 / 保存
  - 支持 PNG / JPG / JPEG / BMP / TIFF / WebP
  - 保存时自动补扩展名
  - 默认文件名：原文件名 + `_export`

- 缩放
  - 右侧有缩放滑块（10% ~ 300%）
  - 预览区支持滚轮缩放
  - 下方状态栏显示当前缩放百分比

- 旋转
  - 角度输入（-360° ~ 360°）
  - 一键旋转 90°
  - 旋转实时预览

- 裁剪
  - 预设比例：自由 / 1:1 / 4:3 / 16:9 等
  - 鼠标框选裁剪区域
  - 半透明矩形预览
  - 支持精确输入 左 / 上 / 宽 / 高

- 文字水印
  - 自定义文本
  - 位置：四角 / 中间 / 自定义坐标
  - 字号可调（px）
  - 透明度可调（0% ~ 100%）
  - 系统字体选择器（调用系统字体列表）
  - 实时水印预览

- 滤镜
  - 无滤镜
  - 黑白
  - 模糊（可调半径）
  - 复古（棕褐色调）
  - 采用 Pillow 实现，效果比较柔和

- 批量处理
  - 入口在菜单栏「工具 → 批量处理」
  - 指定输入文件夹
  - 可选：
    - 统一格式（PNG / JPG / BMP 等）
    - 统一宽度（等比缩放）
    - 统一命名前缀
  - 进度条+可取消
  - 遇到坏图会跳过并记录

- 放映功能
  - 选择文件夹后顺序播放图片
  - 支持预览区播放或全屏播放
  - 可设置播放间隔
  - 支持简单切换动画
  - 支持导出为视频（适合做一个图片轮播小视频）

- 撤销 / 重做
  - 完整的历史栈
  - 撤销 / 重做按钮
  - 常用快捷键支持（如 Ctrl+Z / Ctrl+Y）

- 主题
  - 浅色主题
  - 深色主题
  - 跟随系统
  - 所有对话框尽量保持风格统一

- 一键重置
  - 左侧工具栏最底部「重置」按钮
  - 清空所有操作
  - 回到图片刚打开时的样子
