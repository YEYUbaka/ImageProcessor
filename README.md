# 简易图片处理工具

一个小而实用的图片处理软件。  
不想学 PS，又不想凑合用系统自带的，就用它。

---

## 项目简介

这个项目是用 Python + PyQt6 写的桌面图片工具。  
重点不在炫技，而是在好用。

打开一张图。  
调一调参数。  
看着预览满意了。  
点一下保存。  
事情就做完了。

适合这些场景：

- 给简历、作业、报告准备图片
- 发朋友圈前简单修图
- 给作品加水印，防止乱用
- 批量处理一堆截图或相片
- 在电脑上放映一整个文件夹的照片

---

## 功能一览

只讲真正可用的功能。

### 基础功能

- 打开和保存图片  
  支持常见格式：PNG、JPG、JPEG、BMP、TIFF、WebP。  
  保存时会自动补上扩展名。  
  默认文件名会在原名后面加 `_export`。

- 缩放  
  右侧有缩放滑块，范围 10% 到 300%。  
  预览区可以用鼠标滚轮缩放。  
  状态栏会显示当前缩放比例。

- 旋转  
  输入角度即可，范围 -360° 到 360°。  
  左侧还有“旋转 90°”的快捷按钮。  
  旋转效果支持即时预览。

- 裁剪  
  支持自由裁剪。  
  也支持常见比例，比如 1:1、4:3、16:9 等。  
  可以用鼠标在预览区框选。  
  也可以在右侧输入 左、上、宽、高。  
  裁剪区域是半透明矩形，很直观。

### 进阶功能

- 文字水印  
  输入一行文字。  
  选择位置：四个角、中心、自定义坐标。  
  调字号。  
  调透明度。  
  可用系统字体选择器挑字体。  
  预览区会实时显示水印效果。

- 滤镜  
  内置几种简单滤镜：  
  无滤镜 / 黑白 / 模糊 / 复古。  
  模糊可以调强度。  
  滤镜效果是用 Pillow 做的，偏柔和，不刺激眼睛。

- 批量处理  
  在菜单栏 “工具 → 批量处理”。  
  挑一个输入文件夹。  
  可以统一格式，比如全部转成 JPG。  
  可以统一宽度，保持比例缩放。  
  可以加统一前缀重命名。  
  带进度条，出错会跳过并提示。

- 图片放映  
  挑一个文件夹。  
  可以在预览区播放，也可以全屏播放。  
  可以设置播放间隔。  
  支持几种简单的切换动画。  
  可以把整场放映导出成一个视频文件。

### 界面相关

- 预览区  
  类似 PS 的预览方式。  
  鼠标滚轮缩放。  
  按住左键拖拽移动视图。  
  缩放后裁剪区域会自动适配。  
  下方实时显示缩放百分比。

- 撤销 / 重做  
  每次操作都会入栈。  
  支持撤销和重做。  
  工具栏有按钮。  
  快捷键也支持。

- 主题  
  支持浅色主题。  
  支持深色主题。  
  也可以跟随系统。  
  主窗口和各类对话框的风格统一。

- 一键重置  
  左侧工具栏最下面有个“重置”按钮。  
  一点就把所有操作清空。  
  回到图片刚打开时的样子。
